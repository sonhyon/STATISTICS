<summary>6.1 실험으로 효과를 확인한다 (일원배치 분산분석) </summary>
분산분석의 특징
-그룹을 만드는 처리조건이나 수준이3개 이상인 경우에도평균의 차이를 검정할 수 있다(t검정은 두그룹의 경우에)
-여러 요인의 교호작용을 검정할 수 있다

일원배치 분산분석
-요인(인자)이하나이며 대응은 없는 가장 기본적인 분산분석
-인위적으로 처리조건(수준)을 변화시킨 요인이 실험결과에 영향을 미치는지 확인
-대조군 설정
#처리하지 않은 실험군이며, 대조실험의 기본적인 요건이다
-실험의 반복
#데이터가 분산이 되어있지 않으면 분석을 할 수 없다

분산분석
-총변동: 데이터 전체의 분산
-군간변동: 목적이 되는 요인의 효과에 의한 분산
-군내변동: 목적 이외의 요인인오차의 효과에 의한 분산
-총변동 = 군간변동+군내변동

총변동의 계산
-변동: 편차제곱의 합
-각 값과 총평균의 편차를 제곱해서 모두 더한다

군간변동의 계산
-오차의 효과가 없으면 각 집단 내의 값은 같아진다
-군간변동의 분산은 검정통계량(F값)의 분자가 된다

군내변동의 계산
-본래 같아야 할 집단 내의 값이 흩어져 있는 것은 오차효과가 있기 때문이다
-군내변동 분산은 검정통계량(F값)의 분모가 된다

검정통계량(F값)
F값 = 군간변동의 불편분산 / 군내변동의 불편분산

가설
-귀무가설: 그룹 간의 모평균에는 차이가 없다 -> 요인효과는 없다
-대립가설: 그룹 간의 모평균에는 차이가 있다 -> 요인효과가 있다

<summary> 6.5 검정을 반복해서는 안 된다 (다중성)</summary>

**다중성**  
하나의 연구에서 여러 번 가설검정을 수행하는 상황

**예시**  
집단 A, B, C, D가 있을 때  
A vs B, A vs C, A vs D  
B vs C, B vs D, C vs D  
→ 총 **6번의 검정**

이처럼 여러 번 검정을 반복하면 문제가 발생한다.

**왜 검증을 반복하면 안 될까?**  
핵심 이유는 **제1종 오류(Type I error)** 때문이다.

**제1종 오류**  
- 실제로 차이가 없는데, 있다고 잘못 판단하는 오류  
- 보통 유의수준 α = 0.05 (5%)

**검정을 여러 번 하면**  
- 1번 검정 → 오류 확률 0.05  
- 6번 검정 → 1 − (1 − 0.05)<sup>6</sup> ≈ 0.26  

➡️ 약 **26% 확률**로 거짓 유의미 결과 발생
